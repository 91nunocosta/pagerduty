# PagerDuty API

PagerDuty client API for the interview.

## Installation

```bash
pip install pagerduty
```

## Usage

```python
>>> from pagerduty import fib
>>> fib(0)
0

```

For more details, read the
[documentation](https://91nunocosta.github.io/pagerduty/pagerduty.html).
